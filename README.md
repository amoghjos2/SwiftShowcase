# SwiftShowcase

## Requirements

- Use the [Perenual API](https://perenual.com/docs/api) to fetch plant data.
- Implement a two-screen iOS application:
  - **First screen**: List of plant species.
  - **Second screen**: Detailed view of the selected species.
